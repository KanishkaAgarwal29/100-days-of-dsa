# Day 10 – Array Optimization

## Problems Solved
1. Trapping Rain Water  
2. Product of Array Except Self  
3. Max Consecutive Ones  
4. Third Maximum Number  

## Approach
- Used prefix and suffix based optimization techniques  
- Applied efficient traversal and value tracking  
- Focused on reducing unnecessary computations  

## Time Complexity
- Most solutions run in O(n)

## Key Learning
Many array problems can be optimized efficiently by precomputing values and maintaining running information during traversal.
