# Day 24 – Frequency Maps

## Problems Solved
1. Top K Frequent Elements  
2. Uncommon Words from Two Sentences  
3. Frequency of Array Elements  
4. Find All Numbers Disappeared in an Array  

## Approach
- Applied HashMap and HashSet based techniques  
- Used frequency counting and presence tracking methods  
- Focused on optimizing lookup and duplicate-related problems  

## Time Complexity
- Most solutions run in O(n)

## Key Learning
Frequency maps help efficiently track occurrences, optimize lookups, and simplify many searching and counting problems.
