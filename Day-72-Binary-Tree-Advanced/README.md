# 🌳 Day 72 – Advanced Binary Tree Problems

## Problems Solved

1. Construct Binary Tree from Preorder and Postorder Traversal
2. Construct String from Binary Tree
3. Binary Tree Maximum Path Sum
4. Maximum Level Sum of a Binary Tree

---

## Key Concepts

- Recursion
- Tree Construction
- DFS
- Postorder Traversal
- BFS
- Level Order Traversal
- Global Maximum
- Tree Properties

---

## Key Learning

Today's problems covered different ways of working with binary trees. Tree construction using preorder and postorder traversal strengthened recursive range management, while the maximum path sum problem focused on postorder DFS and tracking a global answer.

The level-order problem reinforced how BFS can be used to process a tree level by level and compare values across different levels.

## Patterns Practiced

- Recursive Tree Construction
- DFS
- Postorder Traversal
- BFS
- Level Order Traversal
- Global Maximum Tracking

## Complexity

| Problem | Time | Space |
|---|---|---|
| Construct Binary Tree from Preorder and Postorder | O(n) | O(n) |
| Construct String from Binary Tree | O(n) | O(h) |
| Binary Tree Maximum Path Sum | O(n) | O(h) |
| Maximum Level Sum of a Binary Tree | O(n) | O(w) |

> `h` = height of the tree  
> `w` = maximum width of the tree

---

## Takeaway

Today's problems strengthened the ability to choose the correct traversal strategy based on the requirement. Whether constructing a tree, calculating a path-based answer, or comparing values level by level, understanding the role of DFS and BFS is essential for solving binary tree problems efficiently.
