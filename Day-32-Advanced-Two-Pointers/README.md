# Day 32 – Advanced Two Pointers

## Problems Solved
1. Boats to Save People
2. Container With Most Water
3. Number of Subsequences That Satisfy the Given Sum Condition
4. Sum of Square Numbers
5. Find the Duplicate Number

## Approach
- Applied two pointer techniques on sorted arrays and value ranges
- Used greedy pairing and search-space reduction strategies
- Practiced counting valid combinations using sorted data
- Explored how pointer movement changes depending on the problem constraints

## Time Complexity
- Most solutions run in O(n) or O(n log n) due to sorting

## Key Learning
Two pointer problems are not limited to simple left-right traversal. The same pattern can be used for greedy pairing, counting valid ranges, mathematical observations, and optimizing brute-force approaches. Understanding *why* a pointer moves is more important than memorizing a specific template.

## Patterns Practiced
- Opposite Direction Pointers
- Sorting + Two Pointers
- Greedy + Two Pointers
- Counting with Two Pointers
- Value-Based Search using Pointers

## Takeaway
Today's problems showed how versatile the two pointer technique is. Although the implementations looked different, each solution relied on reducing the search space intelligently instead of checking every possible combination.
