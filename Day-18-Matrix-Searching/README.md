# Day 18 – Matrix Searching

## Problems Solved
1. Search a 2D Matrix  
2. Search a 2D Matrix II  
3. Leftmost Column With At Least A One 
4. Search in Row-wise Sorted Matrix  

## Approach
- Applied Binary Search and traversal techniques in matrices  
- Used row-wise and column-wise sorted properties for optimized searching  
- Focused on reducing search complexity efficiently  

## Time Complexity
- Most solutions run in O(log n) or O(n + m)

## Key Learning
Matrix searching problems become efficient when row-wise and column-wise ordering properties are utilized properly.
