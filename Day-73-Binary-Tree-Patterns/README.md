## Key Learning

Today's problems focused on important recursive patterns in binary trees. From calculating tree depth and comparing two trees to finding the Lowest Common Ancestor and flattening a tree into a linked list, each problem reinforced a different approach to solving tree problems.

## Complexity

| Problem | Time | Space |
|---|---|---|
| Maximum Depth of Binary Tree | O(n) | O(h) |
| Flatten Binary Tree to Linked List | O(n²) | O(h) |
| Lowest Common Ancestor | O(n) | O(h) |
| Same Tree | O(n) | O(h) |

> `h` = height of the tree.

## Takeaway

Today's problems strengthened my understanding of how recursion can be applied to different binary tree patterns. Understanding the structure of a tree and choosing the right recursive approach is essential for solving a wide range of tree problems efficiently.
