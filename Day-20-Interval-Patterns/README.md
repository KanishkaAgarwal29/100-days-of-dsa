# Day 20 – Interval Patterns

## Problems Solved
1. Insert Interval  
2. Meeting Rooms  
3. Remove Covered Intervals  
4. Interval List Intersections  

## Approach
- Applied interval sorting and overlap handling techniques  
- Used range comparisons and interval traversal  
- Focused on insertion, intersections, and interval optimization logic  

## Time Complexity
- Most solutions run in O(n log n)

## Key Learning
Interval problems become easier after sorting, which helps efficiently handle overlaps, insertions, and intersections.
