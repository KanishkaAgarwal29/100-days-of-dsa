# Day 13 – Binary Search Variations

## Problems Solved
1. Find First and Last Position of Element in Sorted Array  
2. Count Occurrences in Sorted Array  
3. Find Peak Element  
4. Single Element in a Sorted Array  

## Approach
- Applied boundary-based binary search techniques  
- Used pattern observation and index properties  
- Focused on reducing search space efficiently  

## Time Complexity
- Most solutions run in O(log n)

## Key Learning
Binary Search can be extended beyond direct searching by using boundary conditions, index observations, and monotonic patterns.
