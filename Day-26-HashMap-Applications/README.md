# Day 26 – HashMap Applications

## Problems Solved
1. Isomorphic Strings  
2. Largest Subarray with 0 Sum  
3. Word Pattern  
4. Check If N and Its Double Exist  

## Approach
- Applied HashMap and HashSet based optimization techniques  
- Used efficient lookup and mapping strategies  
- Focused on prefix sums, relationship mapping, and fast searching methods  

## Time Complexity
- Most solutions run in O(n)

## Key Learning
HashMaps help optimize many problems efficiently by enabling fast lookups, prefix-based computations, and relationship tracking between elements.
