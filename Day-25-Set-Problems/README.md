# Day 25 – Set Problems

## Problems Solved
1. Next Happy Number  
2. Union of Two Arrays  
3. Contains Duplicate II  
4. Find the Difference of Two Arrays  

## Approach
- Applied HashSet and HashMap based techniques  
- Used presence checking and duplicate handling methods  
- Focused on optimizing searching and uniqueness-related problems  

## Time Complexity
- Most solutions run in O(n)

## Key Learning
Set-based approaches help efficiently handle uniqueness, presence tracking, and duplicate-related problems while reducing unnecessary comparisons.
