# Day 37 – Sliding Window Basics

## Problems Solved
1. Substrings of Size Three with Distinct Characters
2. Defuse the Bomb
3. Maximum Average Subarray I
4. Maximum Number of Vowels in a Substring of Given Length

## Approach
- Practiced fixed-size sliding window techniques
- Maintained a running window instead of recalculating values repeatedly
- Applied add-and-remove logic while moving the window
- Focused on understanding how window boundaries change during traversal

## Time Complexity
- Most solutions run in O(n)

## Key Learning
Sliding Window helps optimize problems involving continuous subarrays or substrings by maintaining useful information inside a moving range instead of recomputing it from scratch.

## Patterns Practiced
- Fixed-size window
- Running sum
- Character counting
- Circular window basics
