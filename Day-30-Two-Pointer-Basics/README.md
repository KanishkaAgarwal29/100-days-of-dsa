# Day 30 – Two Pointer Basics

## Problems Solved
1. Remove Duplicates from Sorted Array II  
2. Check for Subsequence  
3. Special Array Reversal  
4. Apply Operations to an Array  

## Approach
- Practiced slow-fast and opposite-direction pointer techniques  
- Applied traversal and in-place array manipulation strategies  
- Focused on improving pointer movement intuition and optimization-based thinking  

## Time Complexity
- Most solutions run in O(n)

## Key Learning
Two pointer techniques help optimize traversal-based problems efficiently by coordinating pointer movement instead of repeatedly scanning arrays or strings.
