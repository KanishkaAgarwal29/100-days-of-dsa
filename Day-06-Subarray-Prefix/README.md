# Day 6 – Subarray (Prefix Sum + Hashmap)

## Problems Solved
1. Subarray Sum Equals K  
2. Subarray with Given Sum  
3. Longest Subarray with Sum K  
4. Count Number of Subarrays with Given Sum  

## Approach
- Used prefix sum to calculate cumulative sums  
- Used hashmap to store previous sums  
- Checked for required sum using (currentSum - target)  

## Time Complexity
- O(n)

## Key Learning
Prefix sum combined with hashmap helps efficiently solve subarray problems involving sum conditions.
