# Day 2 – Array Manipulation

## Problems Solved

1. Move Zeroes
2. Remove Duplicates from Sorted Array
3. Merge Sorted Array
4. Left Rotation

## Approach

* Practiced modifying arrays in-place
* Used two-pointer technique for efficient operations
* Focused on minimizing extra space usage

## Time Complexity

* Most solutions run in O(n)

## Key Learning

Understanding in-place operations and efficient array manipulation is important for optimizing both time and space complexity.
