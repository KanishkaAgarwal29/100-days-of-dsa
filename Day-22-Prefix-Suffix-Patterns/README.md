# Day 22 – Prefix-Suffix Patterns

## Problems Solved
1. Product of the Last K Numbers
2. Left and Right Sum Differences
3. Find the Highest Altitude
4. Minimum Value to Get Positive Step by Step Sum

## Approach
- Practiced prefix-based accumulation techniques
- Used running sums and products for efficient computation
- Focused on avoiding repeated calculations through precomputed values

## Time Complexity
- Most solutions run in O(n)

## Key Learning
Prefix and suffix patterns help optimize problems by maintaining running information instead of recalculating values repeatedly.
