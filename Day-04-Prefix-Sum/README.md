# Day 4 – Prefix Sum

## Problems Solved
1. Running Sum of 1D Array  
2. Find Pivot Index  
3. Range Sum Query – Immutable  
4. Subarray Sum Equals K  

## Approach
- Used prefix sum technique to compute cumulative sums  
- Applied it for range queries and subarray problems  
- Optimized brute force solutions  

## Time Complexity
- Most solutions: O(n)

## Key Learning
Prefix sum helps reduce time complexity in range and subarray problems by avoiding repeated calculations.
