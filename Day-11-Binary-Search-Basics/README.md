# Day 11 – Binary Search Basics

## Problems Solved
1. Binary Search  
2. Search Insert Position  
3. Guess Number Higher or Lower  
4. Find Smallest Letter Greater Than Target  

## Approach
- Applied binary search to reduce search space efficiently  
- Used low, high, and mid pointers carefully  
- Focused on search conditions and boundary updates  

## Time Complexity
- Most solutions run in O(log n)

## Key Learning
Binary Search works by repeatedly dividing the search space in half, making search operations highly efficient on sorted data.
