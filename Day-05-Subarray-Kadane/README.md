# Day 5 – Subarray (Kadane)

## Problems Solved
1. Maximum Subarray  
2. Maximum Sum Circular Subarray  
3. Maximum Product Subarray  
4. Maximum Subarray Sum (GFG)  

## Approach
- Used Kadane’s Algorithm for maximum subarray problems  
- Extended it for circular arrays  
- Tracked both min and max for product-based cases  

## Time Complexity
- O(n)

## Key Learning
Kadane’s Algorithm helps decide whether to extend the current subarray or start a new one for optimal results.
