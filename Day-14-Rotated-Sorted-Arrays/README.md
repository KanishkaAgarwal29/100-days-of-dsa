# Day 14 – Rotated Sorted Arrays

## Problems Solved
1. Search in Rotated Sorted Array  
2. Find Minimum in Rotated Sorted Array  
3. Search in Rotated Sorted Array II  
4. Rotation Count in Rotated Sorted Array  

## Approach
- Applied Binary Search on partially sorted arrays  
- Identified sorted halves to reduce search space  
- Used observation-based conditions for efficient searching  

## Time Complexity
- Most solutions run in O(log n)

## Key Learning
Binary Search can still be applied efficiently on rotated arrays by carefully identifying the sorted portion at each step.
