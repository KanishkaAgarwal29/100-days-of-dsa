# 🌳 Day 74 – Binary Tree Recursion

## Problems Solved

1. Children Sum in a Binary Tree
2. Root to Leaf Paths
3. Identical Trees
4. Sum of Left Leaf Nodes

---

## Key Concepts

- Recursion
- DFS
- Backtracking
- Root-to-Leaf Traversal
- Tree Comparison
- Leaf Node Identification
- Tree Properties

---

## Key Learning

Today's problems focused on applying recursion to different binary tree patterns. The problems covered validating a property of a tree, generating all root-to-leaf paths using backtracking, comparing two trees structurally, and identifying specific leaf nodes during DFS traversal.

---

## Patterns Practiced

- Recursive DFS
- Backtracking
- Root-to-Leaf Traversal
- Tree Comparison
- Conditional Tree Traversal

---

## Complexity

| Problem | Time | Space |
|---|---|---|
| Children Sum in a Binary Tree | O(n) | O(h) |
| Root to Leaf Paths | O(n) | O(n) |
| Identical Trees | O(n) | O(h) |
| Sum of Left Leaf Nodes | O(n) | O(h) |

> `h` = height of the tree.

---

## Takeaway

Today's problems reinforced how recursion can be adapted to solve different tree-related tasks. From validating relationships between nodes to collecting complete root-to-leaf paths, recognizing the correct base case and recursive structure remains the key to solving binary tree problems efficiently.
