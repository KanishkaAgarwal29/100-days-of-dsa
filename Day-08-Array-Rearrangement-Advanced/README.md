# Day 8 – Array Rearrangement (Advanced)

## Problems Solved
1. Sort Colors  
2. Rearrange Array Alternately (Max-Min Form)  
3. Partition Array According to Pivot  
4. Wiggle Sort  

## Approach
- Used multi-pointer techniques for partitioning  
- Applied index-based rearrangement strategies  
- Focused on in-place modification and efficient swapping  

## Time Complexity
- Most solutions run in O(n)

## Key Learning
Advanced rearrangement problems require careful index handling and pointer management to maintain order efficiently.
