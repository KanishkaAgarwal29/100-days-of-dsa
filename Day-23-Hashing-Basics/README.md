# Day 23 – Hashing Basics

## Problems Solved
1. Valid Anagram  
2. Contains Duplicate  
3. First Repeating Element  
4. Longest Consecutive Sequence  

## Approach
- Applied hashing and frequency counting techniques  
- Used HashMap and HashSet for efficient lookups  
- Focused on optimizing searching and duplicate handling problems  

## Time Complexity
- Most solutions run in O(n)

## Key Learning
Hashing helps optimize many problems efficiently by enabling fast lookups, frequency tracking, and duplicate detection.
