# Day 17 – Matrix Traversal

## Problems Solved
1. Spiral Matrix  
2. Spiral Matrix II  
3. Toeplitz Matrix  
4. Shift 2D Grid  

## Approach
- Practiced boundary-based and directional matrix traversal  
- Applied row, column, and diagonal movement techniques  
- Focused on matrix pattern observation and index handling  

## Time Complexity
- Most solutions run in O(m × n)

## Key Learning
Matrix traversal problems require careful handling of boundaries, directions, and indexing to avoid revisiting elements and maintain correct movement patterns.
