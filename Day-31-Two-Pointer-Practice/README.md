# Day 31 – Two Pointer Practice

## Problems Solved
1. K-diff Pairs in an Array
2. Sort Array By Parity
3. Assign Cookies
4. Maximum Matching of Players With Trainers
5. Append Characters to String to Make Subsequence

## Approach
- Practiced two pointer techniques on arrays and strings
- Applied sorting-based pointer movement and greedy strategies
- Used pointer coordination to efficiently match, compare, and traverse elements
- Focused on understanding when to move one pointer versus both pointers

## Time Complexity
- Most solutions run in O(n) or O(n log n) depending on sorting requirements

## Key Learning
Two pointer problems are not just about maintaining two indices—they are about making efficient decisions that reduce the search space. Today's problems demonstrated how sorting, greedy choices, and pointer movement often work together to produce optimal solutions.
